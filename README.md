# PurpleBlueTempConvert

# مبدل دما - سلسیوس به فارنهایت

یک ابزار ساده و زیبا برای تبدیل دما از **سلسیوس** به **فارنهایت** (و برعکس) با استفاده از HTML، CSS و JavaScript خالص.



## ویژگی‌ها
- طراحی مدرن با گرادیانت پس‌زمینه (Purple-Blue theme)
- ورودی و خروجی realtime (با تایپ عدد، نتیجه فوری نشون داده می‌شه)
- دکمه‌های زیبا و responsive
- پاک کردن ورودی با یک کلیک
- پشتیبانی از زبان فارسی


- HTML5
- CSS3 (Flexbox + Gradients + Animations)
- Vanilla JavaScript
